# 2023_LineTracer

2023 RATS 신입생 라인트레이서 프로젝트

## 라인트레이서 대회
- 라인트레이서 완성 후 라인트레이서 대회에서, 준회원에서 정회원으로 승급됩니다.
    - 대회에서 여러 번 도전 가능
    - 최소 1회 완주 성공해야 정회원 승급 가능
- 정회원이 되어야 RATS에서 계속 활동할 수 있습니다.
